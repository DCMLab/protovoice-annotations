Protovoice Annotations

On this page, you can find a collection of protovoice analyses. You can explore them by clicking on individual nodes in the graph and seeing how they are connected to the rest of the score, or by walking through a derivation step by step. The analysis files can be found on GitHub.

If you want to create your own analyses, check out the annotation tool. To view analyses that are not in this repository, you can use the protovoice viewer. The source code for both tools is available on GitHub as well.
